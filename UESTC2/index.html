UESTC “4750”的传奇

Overview

刚刚学数位dp找点题鏼鏼

题意

$[L, R]$中$4750$出现了几次

题解

$f_{pos, state}$表示填到$pos$位，目前与$4750$匹配到$state$位，<答案，到目前为止有多少数>。

直接记忆化搜索即可，dfs(pos, state, lim, a)表示填到$pos$，状态是$state$，是否为最高位，上界的答案，然后对于$state$定义一个nxt[]={4, 7, 5, 0}进行转移，失配的时候转移到i==4即可。本地对拍跑了几组都是Accepted，不过似乎有点爆栈？

一开始没有加if (s.size() < 4) return 0;直接崩了，然后就去看HeRaNO的做法，并没有很好地理解就去问了NWPU_zhaoyiping，然后他用了他昨天教我的写法写，发现和我第一次写出来的并没有不同，除了有一个特判……
