E. Matching vs Independent Set

题意

$3n$个点$m$条边，求一个大小是$n$的独立集或者匹配

题解

完全没想到，把所有的边标记一下然后取端点不重合的边，够$n$了就是匹配，然后取没有标记过的点就是独立集。是一个考察定义的好题，但是放在E题不妥当
