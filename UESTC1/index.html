数据结构

UESTC数据结构专题 我的题面最简单，快来做！

Overview

有一天和HeRaNO聊天的时候他突然说一个数据结构题调了他三个小时没调出来，然后恰好那个题被敝校一位潜入UESTC ICPC群里的队员拿出来问，觉得很好，就去做了一下。

题意

题目链接

  • 0 l r a k p：在$[l, r]$这一段加上两个等差数列，以$p - L + 1$为界
  • 1 l r：询问$[l, r]$中最长的等差连续子序列

题解

直接看操作一并不复杂，直接用线段树区间加分两段计算lazy tag就行了，但是直接维护原序列对于操作二来说没什么好方法。
再单独看操作二，对于等差数列的处理一般是差分一次之后找相同的连续一段，或者是差分两次之后找连续的一段0.但是如果是离线的差分操作又不好维护操作一。
因此想到了题解线段树维护差分数组。这里选择两次差分（NWPU_zhaoyiping力荐）：
维护差分数组$cnt_i$，表示位置$i$上的差分标记，对于操作一可以转化成下面几个标记：

  • $cnt_l += a, cnt_{l + 1} -= a$
  • $cnt_{l + 1} += k, cnt_{p + 1} -= k$
  • $cnt_{p + 1} -= k, cnt_{r + 1} += k$
  • $cnt_{r + 1} += -a - k(2p - l - r), cnt_{r + 2} -= -a - k(2p - l - r)$

由于端点有差分标记，因此考虑维护$(l, r)$内的答案，结果加上端点即可。维护连续一段0的方法是分治，分别维护左边一段的答案、右边一段的答案以及整段的答案。
